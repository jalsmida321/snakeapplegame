Operation Instructions

Play the classic snake eating apple game in a pop-up window at any time!
The goal of the game is to consume as many fruits as possible without hitting the four borders of the game area.
The snake needs to move around the board to collect apples that appear randomly on the screen. The player can control the snake's direction by using the arrow keys.
As the snake eats apples, its body gets longer, making it more difficult to navigate through the board without colliding with anything.
The extension can be downloaded & installed from Chrome Web Store: Enjoy it!
